🔒 Privacy Policy

Aapki privacy humari priority hai

Last Updated: January 2025

1. Introduction

Lul Downloader ("hum", "hamari website") aapki privacy ko seriously leta hai. Yeh Privacy Policy batati hai ki hum kaunsa data collect karte hain (ya nahi karte) aur usse kaise use karte hain.

2. Kya Data Collect Karte Hain?

Hum minimum data collect karte hain:

  • Server Logs: Basic access logs (IP address, browser type, request time) — yeh standard web server logs hain jo 30 din mein automatically delete ho jaate hain.
  • Download URLs: Jo TikTok links aap submit karte hain, woh sirf download process ke liye use hoti hain. Hum inhe store nahi karte.
  • Analytics: Hum aggregate traffic data dekhte hain (total visitors, popular pages) — koi personal information nahi.

3. Kya Hum Store Nahi Karte?

  • Aapka naam ya email
  • Aapka location (beyond country level)
  • Download ki gayi videos
  • Aapki download history (yeh sirf aapke browser mein localStorage mein save hoti hai)
  • Koi bhi personal information

4. Cookies

Hum koi tracking cookies use nahi karte. Sirf aapka theme preference (dark/light mode) aur settings browser ke localStorage mein save hote hain — yeh data hamari servers tak kabhi nahi pahuncha.

5. Third Party Services

Hum Google Fonts use karte hain fonts ke liye. Inki apni privacy policy hai. Aur hum TikTok ke public API se videos fetch karte hain.

6. Children's Privacy

Hamari service 13 saal se kam umra ke bachon ke liye nahi hai. Hum jaante-bujhte minors ka data collect nahi karte.

7. Changes

Hum kabhi bhi is policy ko update kar sakte hain. Changes is page par post kar diye jaayenge.

8. Contact

Privacy se related koi sawaal hai? Humse contact karo.
